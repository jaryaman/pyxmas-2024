# `uv` cheatsheet

## `uv add`

Add a package to the project

## `uv lock`

Update the project's lockfile

## `uv sync`

Synchronize the venv with the lockfile

## `uv run`

Run a script with your environment. Alternatively `source .venv/bin/activate`.
